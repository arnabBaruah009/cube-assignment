# Cube Assignment

## Introduction
A single-page application in React and Typescript which shows a customer details portal.

## Deployed Link
[Link](https://cube-assignment-arnab.netlify.app/)

## Installation & Run

- ```git clone https://github.com/arnabBaruah009/cube-assignment.git```
- ```cd cube-assignment```
- ```npm i```
- ```npm run start```

## Technology Stack
- React
- TypeScript
- TailwindCSS
